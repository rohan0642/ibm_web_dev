🛠️ Support Center

🔐 Login & Access Issues

  • I forgot my login ID or password. Use the “Forgot Password” link or contact admin support.
  • I didn’t receive the verification email. Check your spam folder or request a new one.
  • Unable to access the platform. Try clearing browser cache or switch browsers.

🗳 Voting Questions

  • How do I know my vote was submitted? You'll receive an on-screen confirmation.
  • Can I vote more than once? ❌ No. Each user can vote only once.
  • What if I made a mistake while voting? Votes cannot be changed once submitted. Please review before clicking submit.

💻 Technical Support

  • Supported Devices: Works on latest versions of Chrome, Firefox, Edge, Safari.
  • Website not loading properly? Try refreshing the page or using a different browser/device.
  • Website is slow or freezing? May occur due to traffic. Please retry after some time.
